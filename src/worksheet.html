Мои работы

  • Создания высокопродуктивных пастбищных угодий в условиях Севера и Запада Казахстана и их рациональное использования.

Регалии

Участвует в разработке технологических карт - основа бизнес-плана для СХТП (в том числе для доступа к кредитным ресурсам); агросопровождение - консультации, лабораторные исследования, рекомендации для фермеров.

Павлодарская СХОС реализует высококачественные семена пшеницы, ячменя, проса, гречихи, картофеля, многолетних трав. Оказывает технические услуг: услуги по посеву, химпрополке, уборке культур, услуги по очистке семян. Организует опытные участки для испытания семян на полях СХТП.

Темы научных работ

- Внедрение технологии прототипирования на базе научно-производственной лаборатории университета;

- Применение холодных асфальтобетонных смесей на основе МАК-порошка при ремонте дорог в Республике Казахстан.
